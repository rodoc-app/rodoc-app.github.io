 The {\bfseries{main window}} has its {\bfseries{menu}} and two separate {\bfseries{panels}}.\hypertarget{md_help_01texts_2layout_autotoc_md13}{}\doxysection{\texorpdfstring{Menu}{Menu}}\label{md_help_01texts_2layout_autotoc_md13}
{\bfseries{The menu}} is for using the app functions.


\begin{DoxyItemize}
\item Changing the {\bfseries{settings}}
\item Saving and opening {\bfseries{projects}}
\item Editing {\bfseries{templates}}
\item Making {\bfseries{exports}}
\item {\bfseries{Importing}} data
\end{DoxyItemize}\hypertarget{md_help_01texts_2layout_autotoc_md14}{}\doxysection{\texorpdfstring{Panels}{Panels}}\label{md_help_01texts_2layout_autotoc_md14}
The left panel displays the information regarding the active person or so-\/called {\bfseries{proband}}, while the right panel serves for the {\bfseries{family tree}} visualization.\hypertarget{md_help_01texts_2layout_autotoc_md15}{}\doxysubsection{\texorpdfstring{Left Panel}{Left Panel}}\label{md_help_01texts_2layout_autotoc_md15}
The left panel contains the name of the {\bfseries{proband}} at the top, and next to that options to create a new person or delete the current one. Under that, there are several tabs\+:


\begin{DoxyItemize}
\item {\bfseries{General}} serves for changing the default {\bfseries{proband}} information and {\bfseries{tags}}
\item Tabs {\bfseries{relations}}, {\bfseries{events}}, {\bfseries{files}}, {\bfseries{media}}, and {\bfseries{notes}} permits organizing the same-\/named entities
\end{DoxyItemize}\hypertarget{md_help_01texts_2layout_autotoc_md16}{}\doxysubsection{\texorpdfstring{Right Panel}{Right Panel}}\label{md_help_01texts_2layout_autotoc_md16}
The right panel is also internally organized into three tabs where every view is interactive.


\begin{DoxyEnumerate}
\item {\bfseries{Project overview}} is a list of all {\bfseries{entities}}, which can be filtered.
\item {\bfseries{Close family}} is a diagram of all persons in relation to the {\bfseries{proband}}. These persons are separated into three generations.
\item {\bfseries{Tree view}} is a diagram of the {\bfseries{family tree}} showing the so-\/called {\itshape hourglass} shape diagram. The number of generations shown can be configured, it is also possible to zoom in and out the view. 
\end{DoxyEnumerate}
