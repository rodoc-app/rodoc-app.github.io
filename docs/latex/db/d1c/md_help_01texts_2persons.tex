 Each {\bfseries{person}} has a set of values that can be edited\+:


\begin{DoxyItemize}
\item {\bfseries{Titles}} before and after a full name (e.\+g., {\itshape M.\+Sc.} or {\itshape Ph.\+D.})
\item {\bfseries{Name}} (e.\+g., {\itshape John})
\item {\bfseries{Surname}} (e.\+g., {\itshape Swanson})
\item {\bfseries{Birthname}} (e.\+g., {\itshape Smith})
\item {\bfseries{Gender}} (e.\+g., {\itshape Male})
\item {\bfseries{\doxylink{classDate}{Date} of birth}} (e.\+g., {\itshape 23. 2. 2000})
\item {\bfseries{Place of birth}} (e.\+g., {\itshape Prague})
\item Flag for an {\bfseries{alive}} person, otherwise a {\bfseries{date}} and {\bfseries{place of death}}
\end{DoxyItemize}\hypertarget{md_help_01texts_2persons_autotoc_md18}{}\doxysection{\texorpdfstring{Dates}{Dates}}\label{md_help_01texts_2persons_autotoc_md18}
Every {\bfseries{date}} value can actually be\+:


\begin{DoxyItemize}
\item {\bfseries{Unknown}} as a special value for missing information
\item Just a {\bfseries{single date}} (e.\+g., {\itshape 23. 2. 2000})
\item {\bfseries{Range}} of two dates (e.\+g., {\itshape From\+: 23. 2. 2000 to\+: 25. 2. 2000})
\end{DoxyItemize}

Each date can also be associated with an optional {\bfseries{note}} allowing to hold any information intended.\hypertarget{md_help_01texts_2persons_autotoc_md19}{}\doxysection{\texorpdfstring{Tags}{Tags}}\label{md_help_01texts_2persons_autotoc_md19}
{\bfseries{Tag}} is a pair of a {\bfseries{name}} and its {\bfseries{value}}. Each person can be associated with an arbitrary number of such tags. However, they need to be unique. Whenever a new tag name is used for the first time, it becomes globally available within the entire project.
